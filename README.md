## 概要

Charalarm は好きな2次元のキャラクターに起こされたい！という小野の思いを叶えてお金も稼ぐために作る個人開発のモバイルアプリです。
ユーザーはモーニングコールをして欲しいキャラクターをアプリ内で選択しアラームをセットします。
アラームをセットした時間に電話にキャラクターから電話がかかってきて幸せな起床を実現できます。
