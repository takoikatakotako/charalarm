# Charalarmについて

## サービス内容

Charalarm は2次元のキャラクターに起こされたい！というtakoikatakotakoの思いを叶えてお金も稼ぐために作る個人開発のモバイルアプリです。 
ユーザーはモーニングコールをして欲しいキャラクターをアプリ内で選択しアラームをセットします。
アラームをセットした時間に電話に女の子から電話がかかってきて幸せな起床を実現できます。
